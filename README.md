# 해외송금 전환율 개선 임팩트 계산기

해외송금 서비스의 전환율 개선 시나리오를 시뮬레이션하고 비즈니스 임팩트를 계산하는 도구입니다.

## 기능

- 📊 6단계 전환 퍼널 추적 (회원가입 → 송금완료)
- 💡 개선 시나리오 시뮬레이션
- 💰 비즈니스 임팩트 계산 (GMV, 수익 증가)
- 💾 데이터 저장/불러오기 (LocalStorage)
- 📈 실시간 전환율 계산

## 사용 방법

웹 브라우저에서 index.html 파일을 열거나 배포된 사이트에 접속하세요.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
